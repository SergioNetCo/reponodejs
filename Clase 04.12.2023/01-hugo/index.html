1. Museo virtual

Esta fue la primera práctica de el curso donde hicimos un viaje al pasado para introducirnos el el mundo de el diseño web y como ha ido cambiando durante los años. A traves de la aplicación web ArtStep realizamos un museo virtual donde plasmamos la evolución de IBM, desde 1990 hasta lo que es ahora en 2023.

Aquí os dejo el link para ver el museo: Museo virtual IBM

2. Introducción a Hugo

El día 16 de septiembre empezamos a realizar nuestra primera práctica con Hugo donde a través de un PDF vemos de que va Hugo Framework. Resumiendo el tema se basa en un generador de sitios web escrito en .md (Markdown) y sin necesidad de base de datos o servidores de aplicaciones. En esta práctica lo que hicimos fue instalar Hugo, creamos un site, añadimos diferente contenido, lo configuramos y lo publicamos en un hosting gratuito.

Aquí os dejo el link de un ejemplo en un hosting llamado surge (se explica más adelante): Site Hugo

3. Creación de el site

Esta tarea era muy parecida a la anterior pero en vez de tener una guía para realizarlo, fuimos a nuestra bola. Tenías que coger un tema del que iria la web, en este caso me toco de una zona de viajes. El proyecto se enseño en clase, entonces no tengo ningún link. Tambien recuerdo que me dio varios problemas a la hora de subirlo a un host gratuito y por tiempo no se llegó a subir, por eso no puedo proporcionar ningún link. Más tarde aprendimos que había diferentes maneras de desplegar páginas web de manera super sencilla, de eso se trata el siguiente punto...

4. Despliegue en host gratuito:

Despues de haber aprendido que hay diferentes creadores de páginas web estaticas y dinamicas, era hora de subirlo a un host gratuito. Hasta ahora solo había probado con 000webhosting o con Github Pages pero Alfonso, nos enseño que actualmente hay muchisimas maneras de subirlo a través de consola o con aplicaciones gratuitas. En esta práctica hemos usado Netlify y Surge.
