3. ¿Es vulnerable HTTP?

Despues de editar y configurar bastantes ficheros ya habíamos aprendido una gran parte de Apache y sus ficheros. Lo que hicimos despues es montar apache en una máquina virtual donde instalariamos un Ubuntu 20.04 a traves de la terminal y seguido PHP. Esta práctica se basaba en ver que seguridad tenía un servidor web sin el certificado de seguridad HTTPS.

Instalamos apache, php y una serie de ficheros que hacian de login. Con wireshark comprobamos la poca seguridad que tenía http y se podia ver una vez introducias las credenciales el usuario y la contraseña que se introducía.
